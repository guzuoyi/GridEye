# GridEye - 网格化时空聚合高速异常事件检测

基于网格化时空聚合与大小模型协作的高速公路异常事件全生命周期检测方法。

## 测试结果

| 视频 | 帧数 | 事件 | Qwen调用 | 报警次数 | 消警 |
|:--|:--:|:--:|:--:|:--:|:--:|
| test.mp4 | 3001 | 白天/无事件 | **0** | 0 | -- |
| test1.mp4 | 1938 | 夜间/靠边停车 | **4** | 1(违停) | 车已走 |
| test2.mp4 | 757 | 白天/靠边停车 | **1** | 1(违停) | 未触发 |

## 对比实验

| 视频 | FSM(ByteTrack) | +空间冷却 | GridEye(本文) |
|:--|:--:|:--:|:--:|
| test.mp4 | 0 | 0 | **0** |
| test1.mp4 | 42 | 25 | **4** |
| test2.mp4 | 143 | 86 | **1** |

GridEye 较传统 FSM 方法减少 Qwen 调用 **90.5%** 和 **99.3%**，无事件视频零误报。

python run_v7.py
