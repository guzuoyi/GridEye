# GridEye - 网格化时空聚合高速异常事件检测

基于网格化时空聚合与大小模型协作的高速公路异常事件全生命周期检测方法。

## 系统架构

```
视频流(25fps) → 抽帧(5fps) → YOLOv26s + ByteTrack
                                    ↓
                       网格引擎 SpatialTemporalGrid
                        (50×50px 高斯加权累加)
                                    ↓
                    空间聚类 → [count ≥ 120] → Qwen3.5 9B 双帧确认
                                    ↓
                    异常? → 锁定(50帧) → [count < 30] → Qwen 消警确认
```
